### Install

Install the dependencies

	npm install
	
### Usage

Command arguments

	$ node index.js -p [PAGE_NO]
	
Fetch from page to page

	$ node index.js -p [PAGE_NO] -e [PAGE_NO]
	
For example (page 51)

	$ node index.js -p 51
	
For example (page 1, page 2 and page 3)

	$ node index.js -p 1 -e 3
